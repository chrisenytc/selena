Getting Started

Clone selena repo

git clone https://github.com/enytc/selena.git

Enter in selena directory

cd selena

Install dependencies

npm install

Configure the settings in app/config

Build Selena

gulp build

Run selena

npm start

Test your selena app

npm test

Using Yeoman Generator

Install Selena.js generator

npm install -g generator-selena

Run Selena.js

yo selena
